1. Adopt full agentic architectures (not just chat + tools)
Anthropic’s own 2026 reports and product releases show the winning pattern is autonomous, long-running, multi-agent systems:
	•	Hierarchical orchestration + reflection loops — High-level planner → specialist sub-agents → reflection/correction step. (Seen in real-world wins at companies like Fountain and CRED using Claude for multi-agent workflows.)
	•	Persistent memory & state — Don’t rely on single-session context. Use external stores (vector DBs, databases, or Claude Artifacts-style persistence) so agents can run for hours/days across API calls.
	•	Test oracles & success criteria — Build automated verification steps (unit tests, benchmarks, human-in-the-loop gates) so agents can self-evaluate like the scientific-computing and coding examples Anthropic is showcasing.
	•	Computer-use / desktop automation readiness — Anthropic’s computer-use tool (beta, with specific headers like computer-use-2025-11-24) lets Claude control mouse/keyboard, run scripts, edit files, and navigate UIs. Design your agents to hand off to this when it becomes GA (expected later 2026). Start testing now in their Docker sandbox.
These patterns already work extremely well with current Claude Opus/Sonnet 4.x models and will scale dramatically with Mythos-class capabilities.
2. Security-by-design (the Glasswing wake-up call)
Mythos Preview autonomously finds zero-days, 16–27-year-old hidden bugs, and full exploit chains in OSes, browsers, kernels, and massive codebases. Your agents (especially if they generate/execute code, use tools, or touch real systems) are exactly the kind of software that will be scanned by similar AI soon.
	•	Bake in proactive vulnerability scanning — Add automated steps that run static analysis, dependency checks, and (when available) Anthropic-style AI code auditing.
	•	Sandbox everything — Tool calls, code execution, file I/O, and any external actions should run in isolated environments (Docker, VMs, or serverless sandboxes with strict permissions).
	•	Secure-by-default agent patterns — Least-privilege tool access, output sanitization, no hard-coded secrets, audit logs for every action, and human oversight gates for high-impact tasks.
	•	Supply-chain & open-source hygiene — If your agents rely on third-party frameworks, watch Anthropic’s recent restrictions (e.g., they’re pushing heavy third-party agent usage off Pro/Max subscriptions to pay-as-you-go). Build modular so you can swap or pay directly via API.
Glasswing partners are already using Mythos Preview defensively; the rest of us should treat this as a preview of what’s coming and harden now.
