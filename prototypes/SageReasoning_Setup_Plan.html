2. How the Stoic Brain Works (Both Access Modes)

MODE A: Reference File (download/read)   |   MODE B: Live API (query/score)

MODE A — Reference File

stoic-brain.json on GitHub
↓
Human downloads / reads
↓
AI agent loads file into context
↓
Both use it for reasoning

Good for: offline use, AI system prompts, research

MODE B — Live API

Stoic Brain in Supabase DB
↓
Website sends query (e.g. "rate this action")
↓
AI agent calls API endpoint
↓
Returns virtue scores + reasoning

Good for: scoring, dashboards, real-time advice
