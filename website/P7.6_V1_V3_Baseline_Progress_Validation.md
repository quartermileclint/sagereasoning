## Tool-by-Tool Comparison

### Baseline Assessment

| Dimension | V1 | V3 |
|---|---|---|
| Question count | 4 core + 1 conditional (Q5) | 5 core + 1 conditional (Q6) |
| Question design | One per independently-weighted virtue | One per progress dimension + passion identification |
| Option scoring | 0-100 primary + secondary numeric scores per virtue | Qualitative dimension level (emerging/developing/established/advanced) |
| Scoring model | Primary (70%) + secondary (30%) blend → virtue weights → total score → tier | Dimension levels → Senecan grade derived from pattern + oikeiosis modifier |
| Primary output | Single 0-100 total score + alignment tier | Senecan grade + 4 dimension levels + dominant passion + oikeiosis stage |
| Conditional question | Q5: triggered by borderline total score (65-74 or 35-44) | Q6: triggered by borderline grade placement (near grade boundaries) |
| Interpretation | Generic text based on strongest/growth virtue | Multi-paragraph text covering strongest/growth dimension + dominant passion description |

**Functional equivalence:** V3 baseline serves equivalent function (assess starting position) with a fundamentally different approach. V1 produced a single number and a tier. V3 produces a multi-dimensional profile: where you stand on each dimension of progress, which passion is most dominant, how broad your concern extends, and which Senecan grade you occupy. The V3 output is more diagnostic and actionable — it tells you specifically what to work on.

V3 tracks multiple dimensions: are passions decreasing? Is proximity improving? The direction-of-travel metric (improving/stable/regressing) replaces the V1 score trend with a more meaningful signal.

## What V3 Adds That V1 Lacked

1. **Multi-dimensional baseline:** V1 reduced the assessment to a single number. V3 maps the user's position across 4 progress dimensions, providing a richer starting profile.

2. **Passion identification in baseline:** V1 had no concept of identifying the user's dominant passion. V3's Q5 identifies which root passion is most active, giving the user a specific focus area.

3. **Oikeiosis assessment:** V1 did not assess circle of concern. V3's Q4 identifies the user's current oikeiosis stage, providing a clear extension target.

4. **Senecan grades:** V1 used arbitrary 5-tier categories (sage/progressing/aware/misaligned/contrary). V3 uses Seneca's source-grounded 3-grade framework (Ep. 75.8-15) plus pre-progress, which is philosophically defensible.

5. **Direction of travel:** V1 tracked a score going up or down. V3 tracks direction across multiple dimensions, providing more nuanced progress feedback.

6. **Passion milestones:** V1 had no milestones related to managing passions. V3 adds 3 passion-related milestones (awareness, zero passions, reduction) that directly track the user's relationship with destructive passions.

7. **Oikeiosis milestones:** V1 had no milestones for expanding concern. V3 adds 2 milestones tracking progress through oikeiosis stages.

---

## What V1 Had That V3 Handles Differently

1. **Single total score:** V1 gave a definitive number (0-100). V3 replaces this with a Senecan grade and dimension levels. The grade provides a clear position; the dimensions provide detail.

2. **Per-virtue scores:** V1 showed 4 virtue bars. V3 replaces these with 4 dimension bars (passion reduction, judgement quality, disposition stability, oikeiosis extension) per R6b — virtues are inseparable, so scoring them independently contradicts the unity thesis.

3. **Virtue-specific milestones:** V1's "Owl's Eye" (wisdom 70+ x3), "Balanced Scales" (justice 70+ x3), etc. are removed. V3 replaces them with proximity and passion milestones that are philosophically grounded and don't require independent virtue scoring.

4. **VirtueConstellation radar chart:** V1 used a radar chart showing 4 virtue scores. V3 removes this because it implies independent virtue assessment (R6b). The proximity distribution bar chart serves the equivalent visualization purpose.

5. **"Prokoptos" milestone name:** V1 used Greek term. V3 renames to "The Path Walked" for journal completion (R8c).
