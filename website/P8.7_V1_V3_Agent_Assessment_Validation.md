## What V3 Adds That V1 Lacked

1. **1:1 journal mapping:** V1 had 37 assessments loosely mapped to 7 journal phases. V3 has 55 assessments each mapping to a specific V3 journal day, ensuring complete coverage of all V3 teachings.

2. **Full passions taxonomy:** V1 assessed "passion-to-eupatheia mapping" in one assessment (TE-05). V3 dedicates an entire phase (Phase 5: 9 assessments) to diagnosing all 4 root passions and their sub-species using the 5-step diagnostic.

3. **Oikeiosis assessment:** V1 mentioned oikeiosis in one assessment (JU-05). V3 dedicates 4 assessments to oikeiosis mapping (RA-02 through RA-04, IN-04) and includes oikeiosis_circles_at_stake in baseline scenarios.

4. **4-stage evaluation sequence:** V1 used a generic scoring rubric (0-100). V3 applies the same 4-stage evaluation sequence used for human actions: prohairesis filter, kathekon assessment, passion diagnosis, unified virtue assessment.

5. **Senecan grade integration:** V1 used 5 arbitrary tiers (sage/progressing/aware/misaligned/contrary). V3 uses Seneca's source-grounded 3-grade framework (Ep. 75.8-15) plus pre-progress.

6. **Dimension-level progress:** V1 tracked a single composite score. V3 tracks 4 dimensions (passion reduction, judgement quality, disposition stability, oikeiosis extension) with qualitative levels.

7. **False judgement specificity:** V1 assessed "specific evidence" generically. V3 requires naming the specific false judgement and providing the correct judgement as remedy for every passion detected.

8. **Personalised correction plan:** V1 had generic improvement_direction per virtue. V3 produces a correction plan pairing each false judgement with its remedy and a specific reasoning change.

9. **Self-examination protocol:** V1 had no personalised protocol. V3's IN-03 produces a 3-question protocol targeting the agent's specific weaknesses.

---

## What V1 Had That V3 Handles Differently

1. **Per-virtue 0-100 scores:** V1's FullAssessmentResult had wisdom: { score: 85, sub_virtues: { discernment: 78 } }. V3 replaces this with dimension levels and proximity — the virtues are assessed as expressions of unified excellence (R6b), not independently scored.

2. **Composite score:** V1 computed a weighted composite. V3 replaces this with Senecan grade and dimension levels — there is no single number.

3. **Sub-virtue numeric scores:** V1's output included per-sub-virtue 0-100 scores (discernment: 78, circumspection: 82). V3 replaces these with qualitative levels (emerging/developing/established/advanced) in Phase 4 assessments.

4. **Alignment tiers:** V1 used sage/progressing/aware/misaligned/contrary. V3 uses Senecan grades (grade_1/grade_2/grade_3/pre_progress) which are philosophically grounded.

5. **Generic scoring rubric:** V1's ASSESSMENT_SCORING_PROMPT used 0-100 criteria. V3's V3_ASSESSMENT_SCORING_PROMPT uses proximity-level criteria (sage_like/principled/deliberate/habitual/reflexive).
